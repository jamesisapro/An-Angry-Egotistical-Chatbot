# An-Angry-Egotistical-Chatbot
An angry, hungry, and egotistical chatbot.

This is an AI project I wanted to try, the bulk of the code was made with Chat-GPT. It has a some human edits for QoL, and proper function, & it uses a JSON Import/Export system. The "AI" is trained based upon the responses you want it to tell you.

This isn't a real AI, but more like a easy to handle/run rule-based system.
